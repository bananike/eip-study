### 문제 2 [유사 기출: 2021년 2회 1과목 — UML 다이어그램 정적/동적 분류 정형 패턴]
다음 중 **UML 정적(구조) 다이어그램**에 해당하지 **않는** 것은?

① **클래스 다이어그램(Class Diagram)** — 시스템의 클래스, 속성, 메서드 및 클래스 간의 관계를 표현.
② **객체 다이어그램(Object Diagram)** — 특정 시점에서 객체들의 인스턴스와 그들 간의 관계를 표현.
③ **컴포넌트 다이어그램(Component Diagram)** — 소프트웨어 컴포넌트의 구조와 의존 관계를 표현.
④ **시퀀스 다이어그램(Sequence Diagram)** — 객체 간 메시지 전달 순서를 시간 흐름으로 표현하는 **동적(행위) 다이어그램**.

### 문제 3 [유사 기출: 2021년 3회 1과목 — GoF 디자인 패턴 분류 정형 패턴]
다음 설명에 해당하는 **GoF 디자인 패턴 분류**로 가장 적절한 것은?

> 객체의 **생성 과정을 캡슐화**하여 객체의 생성과 사용을 분리하는 패턴들의 집합으로, **Singleton, Factory Method, Abstract Factory, Builder, Prototype** 등이 여기에 속한다.

① **생성(Creational) 패턴** — 객체 생성 메커니즘을 다루며, 객체 생성 시점·방법을 캡슐화하는 5가지 패턴.
② **구조(Structural) 패턴** — Adapter, Bridge, Composite, Decorator, Facade, Flyweight, Proxy 등 클래스·객체 결합을 다루는 7가지 패턴.
③ **행위(Behavioral) 패턴** — Observer, Strategy, Command, Iterator, State 등 객체 간 책임 분배·알고리즘을 다루는 11가지 패턴.
④ **아키텍처(Architectural) 패턴** — MVC, Layered, Pipe-Filter 등 시스템 전체 구조를 다루는 거시 단위 패턴(GoF 범위 외).

### 문제 4 [유사 기출: 2021년 1회·2회 1과목 — MVC 아키텍처 패턴 정형 패턴]
**MVC(Model-View-Controller) 아키텍처 패턴**에 대한 설명으로 옳지 **않은** 것은?

① **Model**은 애플리케이션의 **데이터와 비즈니스 로직**을 담당하며, View나 Controller에 의존하지 않는다.
② **View**는 **사용자에게 보여지는 화면(UI)** 을 담당하며, Model의 상태를 시각적으로 표현한다.
③ **Controller**는 **사용자 입력을 받아 Model을 갱신**하고, 결과를 표시할 View를 선택·제어한다.
④ **View는 Model과 직접 강하게 결합**되어야 하며, **Controller를 거치지 않고도** Model의 데이터를 자동으로 변경할 수 있다.

② **수식의 후위 표기법(Postfix) 계산 및 중위→후위 변환** — 연산자 우선순위 처리.
③ **깊이 우선 탐색(DFS: Depth-First Search)** — 그래프·트리 탐색 시 방문 경로 저장.
④ **운영체제의 라운드 로빈(Round Robin) 프로세스 스케줄링** — 시간 할당량(Time Quantum)에 따른 **FIFO 순환 처리(큐 자료구조)**.

### 문제 6 [유사 기출: 2021년 3회 2과목 — 정렬 알고리즘 평균 시간복잡도 정형 패턴]
다음 정렬 알고리즘 중 **평균 시간복잡도가 O(n²)** 인 것은?

① **퀵 정렬(Quick Sort)** — 평균 O(n log n), 최악 O(n²).
② **합병 정렬(Merge Sort)** — 평균·최악 모두 O(n log n).
③ **힙 정렬(Heap Sort)** — 평균·최악 모두 O(n log n).
④ **버블 정렬(Bubble Sort)** — 평균·최악 모두 O(n²), 인접 원소 교환 기반 단순 정렬.

### 문제 7 [유사 기출: 2021년 1회 2과목 — 블랙박스/화이트박스 테스트 분류 정형 패턴]
다음 중 **블랙박스 테스트(Black-box Testing) 기법**에 해당하지 **않는** 것은?

① **동등 분할(Equivalence Partitioning)** — 입력 영역을 동치 클래스로 나누어 대표값으로 테스트.
② **경계값 분석(Boundary Value Analysis)** — 입력 경계 부근(최소·최대·바로 안팎)에서 결함이 자주 발생함을 이용.
③ **원인-결과 그래프(Cause-Effect Graphing)** — 입력 조건과 출력 결과의 논리적 관계를 그래프화하여 테스트 케이스 도출.
④ **기초 경로 시험(Basis Path Testing)** — 프로그램의 **순환 복잡도(Cyclomatic Complexity)** 를 이용해 독립 경로 수를 산출하는 **화이트박스** 기법.

### 문제 9 [유사 기출: 2021년 1회 3과목 — 정규화 단계별 정의 정형 패턴]
**제2정규형(2NF)** 의 조건으로 가장 적절한 것은?

① 모든 도메인이 **원자값(Atomic Value)** 이어야 한다 — 반복그룹·다중값 속성 금지.
② **부분 함수 종속(Partial Functional Dependency)** 을 제거하여, 모든 비주요 속성이 기본키에 **완전 함수 종속**되어야 한다.
③ **이행 함수 종속(Transitive Functional Dependency)** 을 제거하여, X→Y→Z 형태의 추이 종속이 없어야 한다.
④ 릴레이션의 **모든 결정자(Determinant)가 후보키**여야 한다.

### 문제 10 [유사 기출: 2021년 2회 3과목 — DDL/DML/DCL 분류 정형 패턴]
다음 중 **DDL(Data Definition Language)** 에 해당하지 **않는** SQL 명령어는?

① **CREATE** — 테이블·뷰·인덱스 등의 객체를 생성하는 DDL.
② **ALTER** — 기존 테이블의 구조(컬럼 추가·변경·삭제)를 변경하는 DDL.
③ **DROP** — 테이블·뷰 등의 객체를 삭제하는 DDL.
④ **SELECT** — 테이블의 데이터를 조회하는 **DML(Data Manipulation Language)** 명령어.

### 문제 11 [유사 기출: 2021년 3회 3과목 — 트랜잭션 ACID 특성 매칭 정형 패턴]
**트랜잭션의 ACID 특성** 중 다음 설명에 해당하는 것은?

> 트랜잭션 내의 모든 연산은 **전부 수행되어 데이터베이스에 반영되거나, 하나도 수행되지 않은 것처럼 취소(All-or-Nothing)** 되어야 한다.

① **원자성(Atomicity)** — 부분 실행을 허용하지 않으며, 실패 시 UNDO로 모든 변경을 되돌린다.
② **일관성(Consistency)** — 트랜잭션 전·후 데이터베이스가 무결성 제약을 만족하는 상태를 유지한다.
③ **격리성/고립성(Isolation)** — 동시에 수행되는 여러 트랜잭션이 서로 간섭하지 않는다.
④ **지속성/영속성(Durability)** — COMMIT된 결과는 시스템 장애가 발생해도 영구히 보존된다.

### 문제 12 [유사 기출: 2021년 1회·3회 3과목 — 뷰(View) 특성 정형 패턴]
**뷰(View)** 에 대한 설명으로 옳지 **않은** 것은?

① 뷰는 **물리적으로 데이터를 저장하지 않는 가상 테이블**이며, 기본 테이블에 대한 SELECT 결과를 동적으로 보여준다.
② 뷰는 **CREATE VIEW** 문과 **SELECT** 질의를 이용해 정의하며, 사용자에게는 일반 테이블처럼 보인다.
③ 뷰를 통해 **일부 컬럼·일부 행만 노출**하여 데이터 접근 권한을 제한하는 **보안 효과**를 얻을 수 있다.
④ 뷰는 **항상 INSERT·UPDATE·DELETE 등 모든 갱신 연산이 자유롭게 가능**하며, ALTER VIEW로 정의된 컬럼을 자유롭게 변경할 수 있다.

### 문제 16 [유사 기출: 2021년 1회·2회 4과목 — 페이지 교체 vs CPU 스케줄링 분류 정형 패턴]

① **FIFO(First In First Out)** — 메모리에 먼저 적재된 페이지부터 교체하는 페이지 교체 알고리즘.

④ **SJF(Shortest Job First)** — 실행 시간(CPU Burst Time)이 가장 짧은 프로세스를 먼저 처리하는 **CPU 스케줄링** 알고리즘.

### 문제 17 [유사 기출: 2021년 1회 5과목 — XP(eXtreme Programming) 핵심가치 정형 패턴]
**XP(eXtreme Programming)** 의 **5가지 핵심 가치(Core Values)** 에 해당하지 **않는** 것은?

① **의사소통(Communication)** — 개발자·고객 간 지속적이고 빈번한 대화.
② **단순성(Simplicity)** — 지금 필요한 만큼만 단순하게 설계·구현(YAGNI).
③ **피드백(Feedback)** — 짧은 반복 주기로 사용자·테스트의 피드백을 즉시 반영.
④ **문서화(Documentation)** — 모든 의사결정과 산출물을 상세 문서로 남기는 활동을 최우선 가치로 둔다.

### 문제 18 [유사 기출: 2021년 2회 5과목 — 블록체인(Blockchain) 특성 정형 패턴]
**블록체인(Blockchain)** 기술에 대한 설명으로 옳지 **않은** 것은?

① **P2P(Peer-to-Peer) 분산 네트워크**에서 거래 정보(블록)를 참여 노드들이 **분산 저장**한다.
② **작업증명(PoW: Proof of Work), 지분증명(PoS: Proof of Stake)** 등의 **합의 알고리즘**으로 블록의 유효성을 검증한다.
③ 각 블록은 **이전 블록의 해시값(Hash)** 을 포함해 **체인(Chain) 구조**를 형성하므로, 중간 블록의 위변조 시 이후 모든 블록의 해시가 깨진다.
④ **중앙 집중식 거래소(Centralized Authority)** 가 모든 거래를 단독으로 검증·승인하며, 참여 노드들은 거래 원장을 보유하지 않는다.

### 문제 19 [유사 기출: 2021년 3회 5과목 — OSI 7계층 기능 매칭 정형 패턴]
**OSI 7계층** 중 다음 기능을 담당하는 계층은?

> **논리적 주소(IP Address)** 를 이용한 **라우팅(Routing)** 과 **패킷 전달(Packet Forwarding)** 을 담당하며, 대표 프로토콜로 **IP, ICMP, ARP, RIP, OSPF, BGP** 등이 있다.

① **데이터링크 계층(Data Link Layer, Layer 2)** — MAC 주소 기반의 프레임 전송·오류 검출(이더넷, PPP).
② **네트워크 계층(Network Layer, Layer 3)** — IP 주소 기반 라우팅 및 패킷 전달.
③ **전송 계층(Transport Layer, Layer 4)** — 종단 간(End-to-End) 신뢰성 있는 전송(TCP, UDP), 포트 번호.
④ **세션 계층(Session Layer, Layer 5)** — 세션 설정·유지·종료, 대화 제어(반이중·전이중).

### 문제 20 [유사 기출: 2021년 1회·2회 5과목 — 대칭/공개키 암호 분류 정형 패턴]
다음 중 **공개키(비대칭키) 암호 알고리즘**에 해당하지 **않는** 것은?

① **RSA** — 큰 정수의 **소인수분해 문제의 어려움**에 기반한 대표적인 공개키 알고리즘(서명·키교환·암호화 모두 가능).
② **ECC(Elliptic Curve Cryptography)** — **타원곡선 이산대수 문제**에 기반한 공개키 알고리즘으로, RSA 대비 짧은 키로 동일 강도 제공.
③ **ElGamal** — **이산대수 문제**에 기반한 공개키 알고리즘으로, 서명·암호화에 사용.
④ **AES(Advanced Encryption Standard)** — 미국 NIST가 DES를 대체해 채택한 **블록 암호 방식의 대칭키 알고리즘**(키 길이 128/192/256 bit).

### 문제 16

- **페이지 교체 알고리즘**: FIFO·LRU·LFU·**OPT(Optimal·Belady's MIN)**·NUR(Not Used Recently)·**Second Chance**·Clock(시계).
- **CPU 스케줄링**: FCFS·**SJF(Shortest Job First)**·**SRTF**·우선순위·**RR(Round Robin)**·**MLQ/MLFQ**·HRN.
- ④ SJF는 CPU Burst Time 최소 우선 → **CPU 스케줄링** → 정답.
- 함정: "FIFO는 페이지 교체에만 쓰임" → X(CPU 스케줄링의 FCFS와 동일 개념).

### 문제 18
**정답 ④** — 블록체인의 핵심은 **탈중앙화(Decentralization)**.
- ①②③은 모두 블록체인 정의·특성(P2P 분산·합의·해시 체인).

### 문제 19
**정답 ② — 네트워크 계층(Layer 3)**.
- L1 물리: 비트 전송(케이블·허브·리피터).
- L2 데이터링크: **MAC 주소**·프레임·오류 검출(이더넷·PPP·스위치·브리지).
- **L3 네트워크: IP 주소·라우팅·패킷**(IP·ICMP·ARP·RIP·OSPF·BGP·라우터).

### 문제 20
**정답 ④** — 대칭/공개키(비대칭) 분류.
- **공개키(비대칭)**: RSA(소인수분해)·ECC(타원곡선)·**ElGamal**(이산대수)·**Diffie-Hellman**(키교환)·**DSA**(전자서명)·**Rabin**·**Knapsack**.
- **대칭키**:
  - **블록 암호**: **AES**(128/192/256)·DES(56)·3DES·SEED(국산·128)·ARIA(국산·128/192/256)·HIGHT(국산·64)·LEA(국산·128).
  - **스트림 암호**: RC4·LFSR.
- ④ AES는 대칭키 블록 암호 → 공개키 X → 정답.
- 함정: "DSA는 키 교환용" → X(서명 전용·키 교환은 DH).

## 📊 자주 틀리는 함정 5가지 (2021년 출제 정형)

1. **비기능 요구사항 ↔ 기능 요구사항 분류** — "회원가입 입력 항목"·"검색 기능"은 기능, "응답시간 3초"·"가용성 99.9%"·"AES-256 암호화"는 비기능(품질·제약). **"어떻게 동작"(HOW WELL) = 비기능** 키워드로 즉답.
2. **UML 정적 7 vs 동적 7 분류** — **시퀀스·유스케이스·활동·상태·통신·타이밍·상호작용개요 7개는 동적**. **클래스·객체·컴포넌트·배치·복합체구조·패키지·프로파일 7개는 정적**. "객체 다이어그램은 동적"이라는 함정 자주.
3. **DDL·DML·DCL 두음 "정·조·제"** — SELECT를 DDL에 끼워넣거나, TRUNCATE를 DML에 끼워넣는 정형 오답. **TRUNCATE = DDL(Auto-Commit·롤백 불가)**·**DELETE = DML(WHERE 조건·롤백 가능)** 비교 출제 잦음.
4. **페이지 교체(FIFO/LRU/LFU/OPT) ↔ CPU 스케줄링(FCFS/SJF/SRTF/RR/우선순위/HRN)** 영역 혼동 — **SJF·RR은 CPU**, **OPT(Belady)·NUR은 페이지**. FIFO만 양쪽 다 존재(CPU의 FCFS와 동일 개념).
5. **공개키 vs 대칭키 분류** — 대표적 함정: **AES를 공개키**에 끼워넣기, **RSA를 대칭키**에 끼워넣기. 공식 매핑: **RSA·ECC·ElGamal·DH·DSA = 공개키** / **AES·DES·3DES·SEED·ARIA·HIGHT·LEA·RC4 = 대칭키**. **국산 4종(SEED·ARIA·HIGHT·LEA)은 모두 대칭키**.
