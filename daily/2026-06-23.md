**블록체인(Blockchain)** = **분산원장(Distributed Ledger) + P2P 합의(PoW/PoS) + 해시 체인** · **중앙 서버 X** · **위·변조 불가**(블록 해시 연쇄) · 응용: 비트코인·이더리움·스마트 컨트랙트 ③ **메타버스(Metaverse)** = **가상(Meta) + 우주(Universe)** = 현실과 융합된 3D 가상 세계 · VR/AR/MR/XR 기반 · 응용: 로블록스·제페토 ④ **디지털 트윈(Digital Twin)** = **현실 객체·시스템을 가상 공간에 1:1 복제** + **실시간 IoT 센서 동기화** → 시뮬레이션·예측·최적화(스마트팩토리·도시·항공 엔진) ⑤ **매시업(Mashup)** = **2개 이상 웹 서비스 결합**으로 **새로운 서비스 창출**(구글맵 + 부동산 정보 = 부동산 지도 서비스) ⑥ **빅데이터 처리 도구 매트릭스** — **Hadoop** = 분산 처리 프레임워크(에코시스템 총칭) / **HDFS** = 분산 **파일 시스템**(저장) / **MapReduce** = 분산 **연산 모델**(Map: 분할·매핑 → Reduce: 집계) / **Spark** = **In-Memory 기반 고속** 처리(MapReduce보다 100배 빠름) / **Tajo** = **국산** SQL on Hadoop 데이터 웨어하우스 / **Hive** = HiveQL(SQL유사) / **Pig** = Pig Latin 스크립트 / **Sqoop** = **RDB ↔ Hadoop 데이터 전송**(SQL-to-Hadoop) / **Flume** = 로그 수집 / **Zookeeper** = 분산 코디네이션 ⑦ **무선·근거리 통신 분류** — **Zigbee** = 저전력·근거리(IEEE 802.15.4) / **Beacon** = BLE 기반 위치 신호(매장 마케팅) / **Wi-SUN** = 스마트그리드·검침용 무선 메시 / **LTE-M(LTE Cat M1)** = LPWA·중대역(1Mbps) / **NB-IoT(Narrowband IoT)** = LPWA·협대역(저속·저전력·심전계·검침) / **LoRa·SigFox** = 비면허 LPWA / **5G** = eMBB·URLLC·mMTC 3대 시나리오 ⑧ **포그(Fog) vs 엣지(Edge) 컴퓨팅** — 둘 다 **클라우드 부하 분산·지연(Latency) 감소** 목적 / **엣지** = **단말(센서·디바이스) 근접**에서 즉시 처리 / **포그** = **단말과 클라우드 사이 게이트웨이·라우터 계층**에서 처리(엣지보다 상위·범위 넓음) ⑨ **양자컴퓨팅·반도체 신기술** — **큐비트(Qubit)** = **양자 비트**(0과 1 **중첩**) · 양자 컴퓨터 최소 단위 / **Memristor(멤리스터)** = **저항(Resistor) + 메모리(Memory)** = **전류 흐름 기억**하는 **4번째 수동소자**(저항·콘덴서·인덕터 외) / **DRM(Digital Rights Management)** = 디지털 콘텐츠 저작권 보호 ⑩ **영상·통신 서비스 변별** — **OTT(Over-The-Top)** = **인터넷 회선을 통한** 동영상 스트리밍(넷플릭스·유튜브·티빙) — "Top" = 셋톱박스 위 / **N-Screen** = **하나의 콘텐츠를 PC·TV·태블릿·스마트폰 등 N개 화면**에서 끊김 없이 / **IPTV** = **IP망 통한 양방향** TV(KT·SKB·LGU+) / **MVNO(Mobile Virtual Network Operator)** = **알뜰폰**(망 임차 사업자) / **VoIP** = IP 기반 음성 통신.

## 🎯 오늘의 기출 8문제 (Day 22의 2회독 — 변별 각도 변경)

### 문제 1 [유사 기출: 정처기 필기 2020년 3회·2022년 1회·2024년 2회 반복 — 클라우드 4모델 정형 패턴]
**클라우드 컴퓨팅 서비스 모델**에 대한 설명으로 가장 옳지 **않은** 것은?

① **IaaS(Infrastructure as a Service)** — 가상 서버·스토리지·네트워크 등 **인프라 자원**을 제공한다. (예: AWS EC2, MS Azure VM)
② **PaaS(Platform as a Service)** — OS·미들웨어·런타임 등 **개발·실행 플랫폼**을 제공하여, 사용자는 애플리케이션 개발·배포에 집중한다. (예: Google App Engine, Heroku)
③ **SaaS(Software as a Service)** — **완성된 응용 소프트웨어**를 인터넷으로 제공한다. (예: Gmail, Salesforce, MS 365)
④ **BaaS(Blockchain as a Service)** — 사용자가 OS·런타임·하드웨어를 모두 **직접 구축·관리**해야 하는 **온프레미스(On-Premise) 전용** 모델이다.

### 문제 2 [유사 기출: 정처기 필기 2021년 2회·2023년 1회·2025년 1회 반복 — 빅데이터 도구 1:1 매칭 정형 패턴]
**빅데이터 처리 도구**와 그 **주요 역할**의 매칭으로 옳지 **않은** 것은?

① **HDFS(Hadoop Distributed File System)** — Hadoop의 **분산 파일 시스템**으로, 대용량 파일을 블록 단위로 여러 노드에 분산 저장한다.
② **MapReduce** — **Map(분할·매핑) → Reduce(집계)** 단계로 동작하는 **분산 연산 프로그래밍 모델**이다.
③ **Sqoop(SQL-to-Hadoop)** — **관계형 DB(RDB)와 Hadoop(HDFS) 간 데이터 import/export 전송** 도구이다.
④ **Tajo** — **In-Memory 기반 연산**으로 MapReduce보다 약 100배 빠른 **실시간 스트림 처리 엔진**으로, 미국 UC 버클리 AMPLab에서 개발되었다.

### 문제 3 [유사 기출: 정처기 필기 2020년 2회·2022년 3회·2024년 1회 반복 — IT 신기술 정의 매칭 정형 패턴]
다음 **IT 신기술 용어**의 정의로 옳지 **않은** 것은?

① **블록체인(Blockchain)** — **P2P 분산원장** 기술로, 각 블록이 **이전 블록의 해시값을 포함**하여 체인으로 연결되며, 중앙 서버 없이 **위·변조가 매우 어렵다**.
② **디지털 트윈(Digital Twin)** — **현실의 객체·시스템을 가상 공간에 1:1로 복제**하여, **실시간 IoT 센서 데이터와 동기화**해 시뮬레이션·예측·최적화에 활용한다.
③ **매시업(Mashup)** — **2개 이상의 웹 서비스를 결합**하여 새로운 서비스를 창출한다. (예: 구글맵 + 부동산 정보 = 부동산 지도 서비스)
④ **메타버스(Metaverse)** — **사용자의 권한 없이도** PC·모바일 기기의 데이터를 **암호화·인질화**하여 금전을 요구하는 **악성코드의 일종**이다.

### 문제 4 [유사 기출: 정처기 필기 2021년 1회·2023년 3회·2025년 2회 반복 — 무선·근거리 통신 분류 정형 패턴]
**무선·근거리 통신 기술**에 대한 설명으로 가장 옳지 **않은** 것은?

① **Zigbee** — **IEEE 802.15.4** 표준 기반, **저전력·근거리** 무선 통신으로 스마트홈·센서 네트워크에 적합하다.
② **NB-IoT(Narrowband IoT)** — **협대역 LPWA(저전력 광역망)** 로, **저속·저전력·저용량** 사물인터넷 통신(원격검침·스마트미터 등)에 사용된다.
③ **Beacon(비콘)** — **BLE(Bluetooth Low Energy)** 기반 근거리 위치·정보 전달 신호로, 매장 마케팅·실내 위치 측위 등에 활용된다.
④ **Wi-SUN** — **5G 핵심 시나리오 중 URLLC(초저지연·고신뢰)** 표준으로, **자율주행차·원격수술** 등 1ms 이하 초저지연 통신을 위해 ITU가 제정한 광대역 셀룰러 무선 통신 규격이다.

### 문제 5 [유사 기출: 정처기 필기 2020년 1회·2022년 2회·2024년 3회 반복 — 포그·엣지 컴퓨팅 비교 정형 패턴]
**엣지 컴퓨팅(Edge Computing)** 과 **포그 컴퓨팅(Fog Computing)** 에 대한 설명으로 가장 옳은 것은?

① 두 기술 모두 **모든 연산을 중앙 클라우드로 집중**시켜 처리하는 **중앙 집중형** 아키텍처로, 클라우드 부하를 **증가**시킨다.
② **엣지 컴퓨팅**은 **단말(센서·디바이스) 근접부**에서 즉시 처리하여 **지연(Latency)을 최소화**하며, **포그 컴퓨팅**은 **단말과 클라우드 사이의 게이트웨이·라우터 계층**에서 처리하여 엣지보다 **범위가 더 넓다**.
③ **포그 컴퓨팅은 단말 자체에서만** 처리하며, **엣지 컴퓨팅은 클라우드 데이터센터에서만** 처리하는 정반대 개념이다.
④ 두 기술 모두 **무선 통신 규격**의 명칭으로, IEEE 802.15.4 표준의 **하위 프로토콜**이다.

### 문제 6 [유사 기출: 정처기 필기 2021년 3회·2023년 2회·2025년 1회 반복 — 양자컴퓨팅·반도체 신기술 정의 정형 패턴]
다음 **신기술 용어**의 정의로 옳지 **않은** 것은?

① **큐비트(Qubit)** — **양자 컴퓨터의 최소 정보 단위**로, 고전 비트와 달리 **0과 1의 중첩(Superposition) 상태**를 가질 수 있다.
② **Memristor(멤리스터)** — **저항(Resistor) + 메모리(Memory)** 의 합성어로, **전류 흐름의 이력(履歷)을 기억**하는 **4번째 수동소자**(저항·축전기·인덕터에 이은)이다.
③ **DRM(Digital Rights Management)** — **디지털 콘텐츠의 무단 복제·배포를 방지**하는 **저작권 보호 기술**이다.
④ **DRM(Digital Rights Management)** — **양자 키 분배(QKD)** 의 표준 명칭으로, **광섬유 기반 단일광자 전송**을 통해 도청 시 양자상태가 붕괴되어 도청을 즉시 탐지하는 양자 암호 통신 프로토콜이다.

### 문제 7 [유사 기출: 정처기 필기 2020년 4회·2022년 1회·2024년 3회 반복 — 영상·통신 서비스 변별 정형 패턴]
다음 **영상·통신 서비스**에 대한 설명으로 가장 옳지 **않은** 것은?

① **OTT(Over-The-Top)** — **공중 인터넷망**을 통해 영화·드라마·음악 등 **콘텐츠를 스트리밍**으로 제공하는 서비스(넷플릭스·유튜브·티빙)로, 명칭 중 "Top"은 **셋톱박스(Set-Top Box) 위**를 의미한다.
② **N-Screen** — **하나의 콘텐츠를 PC·TV·태블릿·스마트폰 등 N개의 서로 다른 화면**에서 끊김 없이 이어서 시청·이용할 수 있도록 하는 서비스이다.
③ **IPTV** — **IP망을 통해 양방향**으로 TV 방송 콘텐츠를 제공하는 서비스(KT olleh tv·SKB Btv·LG U+ tv)이다.
④ **MVNO(Mobile Virtual Network Operator)** — **이동통신 기지국(BTS)·교환기 등 자체 무선 인프라를 직접 구축·운영**하는 **1차 이동통신 사업자**(SKT·KT·LGU+)를 의미한다.

### 문제 8 [유사 기출: 정처기 필기 2021년 2회·2023년 1회·2025년 2회 반복 — 가짜 약어·유사 발음어 함정 정형 패턴]
다음 IT 신기술 약어 중 **실제로 존재하지 않거나 정의가 잘못 매칭된** 것은?

① **HDFS** = Hadoop Distributed File System — Hadoop 분산 파일 시스템.
② **MVNO** = Mobile Virtual Network Operator — 알뜰폰(이동통신 재판매·임차 사업자).
③ **NB-IoT** = Narrowband IoT — 협대역 저전력 광역(LPWA) IoT 통신.
④ **SBaaS(Software Blockchain as a Service)** = Spark·Blockchain·Storage 3가지를 **단일 API**로 통합 제공하는 **ISO/IEC 표준 클라우드 모델**로, IaaS/PaaS/SaaS와 함께 **공식 4대 클라우드 서비스 모델**에 포함된다.

> 함정 보기 4개의 출처·키워드를 정리해, "이 함정을 봤을 때 어떤 정답 개념으로 즉답해야 하는가" 매핑.

- **"BaaS = 온프레미스 전용"** (문제 1 ④) → **BaaS = Blockchain as a Service = PaaS의 변형(블록체인 인프라 제공)** · **온프레미스의 정반대** = 클라우드형 서비스. 함정 즉답: 모든 *aaS는 클라우드 기반.
- **"Tajo = In-Memory · UC 버클리 · 100배"** (문제 2 ④) → **Spark의 정의**. Tajo는 **국산(고려대·한국전자통신연구원 ETRI) SQL on Hadoop 데이터 웨어하우스**. 함정 즉답: **In-Memory·UC 버클리 = Spark**, Tajo는 국산 SQL.

- **"Wi-SUN = 5G URLLC 광대역 셀룰러"** (문제 4 ④) → Wi-SUN은 **스마트그리드·검침용 무선 메시(802.15.4g)** · 저전력 근거리. 5G URLLC는 별개. 함정 즉답: **Wi-SUN = 검침·메시**, URLLC = 5G 자율주행 별개.
- **"포그 = 단말만 / 엣지 = 클라우드센터만"** (문제 5 ③) → **정반대로 매칭**. 엣지가 단말 근접, 포그는 그 상위 게이트웨이 계층(엣지보다 범위 넓음). 함정 즉답: **엣지 = 단말, 포그 = 중간 계층**.
- **"DRM = 양자 키 분배(QKD)"** (문제 6 ④) → DRM = **저작권 보호**, QKD는 **양자 암호 통신** 별개 약어. 함정 즉답: **DRM = 저작권, QKD = 양자키 분배** 별개.
- **"MVNO = 1차 이동통신 사업자"** (문제 7 ④) → MVNO는 **알뜰폰(망 임차 재판매)**. 1차 사업자는 **MNO(Mobile Network Operator)**. 함정 즉답: **MVNO = Virtual = 가상 = 알뜰폰**, **MNO = 1차**.
- **"SBaaS = ISO/IEC 공식 4대 클라우드 모델"** (문제 8 ④) → **존재하지 않는 가짜 약어**. 공식 3대 클라우드 = IaaS/PaaS/SaaS(+BaaS는 PaaS 변형). 함정 즉답: 처음 보는 약어는 가짜 의심.

### 문제 4

- ② **NB-IoT** = LPWA · 협대역 · 저속·저전력 · 원격검침·스마트미터. ✔
- ③ **Beacon** = BLE 기반 근거리 위치 신호 · 매장 마케팅·실내 측위. ✔
- 정형 함정: **Wi-SUN, LoRa, NB-IoT, Sigfox, Zigbee** 등 비슷한 약어 정의 셔플 + 5G·LTE-M 끼워넣기. **Wi-SUN = 스마트그리드·검침 메시** 1자 외워 즉답.

### 문제 5
**정답 ② 엣지 = 단말 근접 즉시 처리(지연 최소화), 포그 = 단말·클라우드 사이 게이트웨이·라우터 계층 처리(엣지보다 범위 넓음).**

- **포그 컴퓨팅(Fog Computing)** = **엣지보다 한 단계 위·단말과 클라우드 사이의 게이트웨이·라우터·로컬 서버 계층**에서 처리. 시스코(Cisco)가 제안. 엣지보다 **범위가 넓고 자원이 풍부**.
- ① 정반대 — 분산형 처리(엣지/포그)는 클라우드 부하 **감소**. X
- ③ **엣지·포그 정의가 정반대로 뒤바뀜** — 엣지가 단말, 포그가 중간 계층. X
- ④ 엣지·포그는 **무선 통신 규격이 아님** — IEEE 802.15.4와 무관. X
- 정형 함정: **엣지·포그 정의 뒤바꿈** 단골("포그가 단말, 엣지가 게이트웨이"는 정반대). **"엣지(Edge) = 가장자리 = 단말"** / **"포그(Fog·안개) = 단말과 클라우드 사이"** 어원으로 외워 즉답.

### 문제 6
**정답 ④ DRM = 양자 키 분배(QKD) 표준 명칭·광섬유 단일광자 전송·도청 시 양자상태 붕괴.**
- **DRM(Digital Rights Management)** 은 **디지털 콘텐츠 저작권 보호 기술** — 무단 복제·배포 방지·접근 제어·키 관리·DRM 컨테이너(WMV·iTunes Fairplay 등). **양자 통신과 무관**.
- **양자 키 분배(QKD: Quantum Key Distribution)** 는 별개 개념 — **BB84 프로토콜** 등 양자 암호 통신 표준. 도청 시 양자상태(중첩) 붕괴로 탐지 가능.
- ① **큐비트(Qubit)** = 양자 컴퓨터 최소 정보 단위 · **0과 1의 중첩(Superposition)** · 양자 얽힘(Entanglement). ✔
- ② **Memristor** = Memory + Resistor · **전류 흐름 이력 기억** · **4번째 수동소자**(저항·축전기·인덕터 외 + 1971년 추 휘 차일라 박사 이론·2008년 HP 실증). ✔
- ③ **DRM = Digital Rights Management = 저작권 보호**. ✔ (③과 ④는 같은 약어이지만 ④는 정의가 완전히 잘못됨)
- 정형 함정: **DRM = QKD·DRM = 큐비트·DRM = 양자 암호** 같은 가짜 정의 끼워넣기. **DRM = 저작권 보호** 1자 매칭. 같은 약어를 두 번 등장시켜 한 번은 정의 뒤바꾸는 변형 패턴 단골.

### 문제 7
**정답 ④ MVNO = 자체 무선 인프라를 직접 구축·운영하는 1차 이동통신 사업자(SKT·KT·LGU+).**
- **MVNO(Mobile Virtual Network Operator)** = **가상(Virtual) 이동통신 사업자 = 알뜰폰** — **MNO(1차 사업자)로부터 망을 임차·재판매**하는 사업자. **자체 무선 인프라 구축 X**.
- **자체 무선 인프라(기지국·교환기 등)를 직접 구축·운영하는 1차 사업자 = MNO(Mobile Network Operator)** — SKT·KT·LGU+ 3사가 한국의 MNO.
- ① **OTT** = Over-The-Top · **공중 인터넷망** 통한 영상 스트리밍 · 넷플릭스·유튜브·티빙 · "Top" = 셋톱박스 위(셋톱박스 없이 인터넷으로). ✔
- ② **N-Screen** = 하나의 콘텐츠를 N개 화면(PC·TV·태블릿·폰)에서 끊김 없이 이어서 사용. ✔
- ③ **IPTV** = IP망 기반 양방향 TV · KT olleh tv·SKB Btv·LG U+ tv. ✔
- 정형 함정: **MVNO ↔ MNO 정의 뒤바꿈** 단골. **"MVNO = Virtual = 가상 = 알뜰폰(망 임차)"** / **"MNO = 1차 사업자(망 직접 구축)"** 어원으로 외워 즉답. 한국에서는 KT M모바일·LGU+ U+알뜰모바일·SK 7모바일 등이 MVNO.

### 문제 8
**정답 ④ SBaaS(Software Blockchain as a Service) = ISO/IEC 표준 공식 4대 클라우드 서비스 모델.**
- **"SBaaS"는 실제로 존재하지 않는 가짜 약어**. ISO/IEC 17788(클라우드 컴퓨팅 어휘) 등 국제 표준이 인정하는 클라우드 서비스 모델은 **IaaS/PaaS/SaaS 3대 기본 모델 + 파생(BaaS=Blockchain, FaaS=Function, DBaaS=Database 등)**. **SBaaS는 어디에도 등재되지 않음**.
- ① **HDFS = Hadoop Distributed File System**. ✔ (Hadoop 분산 파일 시스템)
- ② **MVNO = Mobile Virtual Network Operator**. ✔ (알뜰폰)
- ③ **NB-IoT = Narrowband IoT**. ✔ (협대역 LPWA IoT)
- 정형 함정: **처음 보는 약어 + 그럴듯한 풀네임 + "ISO/IEC 표준·공식·4대"** 같은 권위 수식어 결합 — **즉시 가짜 의심**. 실존 약어와 1자 다른 변형(NB-IoT → NB-IPv6, BaaS → SBaaS) 단골 함정.

## 📚 역추적 참조 (사후 검증용)
- **클라우드 4모델 (IaaS/PaaS/SaaS/BaaS)**: 정처기 필기 2020년 3회·2022년 1회·2024년 2회 — **BaaS = Blockchain**(시험 기준) · IaaS/PaaS/SaaS/BaaS **제공 범위 1:1 매칭**. "BaaS = 온프레미스·하드웨어 직접 관리" 단골 함정.
- **빅데이터 도구 매트릭스**: 2021년 2회·2023년 1회·2025년 1회 — **Spark = In-Memory·100배·UC 버클리** / **Tajo = 국산·SQL on Hadoop·고려대·ETRI** 정의 뒤바꿈 단골. Sqoop = RDB↔Hadoop 전송.
- **IT 신기술 정의 (블록체인·메타버스·디지털트윈·매시업)**: 2020년 2회·2022년 3회·2024년 1회 — **메타버스 자리에 랜섬웨어·악성코드 정의 끼워넣기** 단골. 정의 셔플로 변별력 평가.
- **무선·근거리 통신 (Zigbee/Beacon/Wi-SUN/NB-IoT/LTE-M)**: 2021년 1회·2023년 3회·2025년 2회 — **Wi-SUN = 5G URLLC**, **NB-IoT = 광대역** 같은 정의 뒤바꿈 단골. Wi-SUN은 스마트그리드 메시(802.15.4g).
- **포그·엣지 컴퓨팅**: 2020년 1회·2022년 2회·2024년 3회 — **엣지·포그 정의 뒤바꿈** 단골. 엣지 = 단말, 포그 = 중간 게이트웨이 계층(시스코 제안).
- **큐비트·Memristor·DRM**: 2021년 3회·2023년 2회·2025년 1회 — **DRM = 양자키 분배(QKD)** 같은 가짜 정의 끼워넣기. Memristor = 4번째 수동소자(추 휘 차일라 1971·HP 2008).
- **OTT·N-Screen·IPTV·MVNO·MNO**: 2020년 4회·2022년 1회·2024년 3회 — **MVNO ↔ MNO 정의 뒤바꿈** 단골. MVNO = Virtual = 알뜰폰.
- **가짜 약어 함정**: 2021년 2회·2023년 1회·2025년 2회 — **SBaaS·DRMaaS·MetaIoT** 같은 가짜 약어 + ISO/IEC·공식·4대 권위 수식어 결합. 처음 보는 약어는 즉시 가짜 의심.
